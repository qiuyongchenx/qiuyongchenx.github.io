邱永臣与世界（J）

Feb 28, 2017

『本篇均为素材，以上』

1.陆莎说：“你发的豆瓣内容，都是关于你自己的，你看我发的东西，大多和客观世界有关。”邱永臣觉得，嗯，自己女朋友真是可爱，能提炼邱永臣豆瓣的核心。也是，自己沉迷心学，喜欢将别人一套一套的哲学思想一概而论，找借口说遵从内心，追随良知，说什么强者度己圣者度人，唉，先努力成为强者吧。

2.妻子卖掉头发给丈夫买表链，丈夫卖掉手表给妻子买发饰。

3.“友谊诚可贵，爱情价更高”，爱情价再高能高得过一张手纸？价值，到底是无价，还是无值，还是无价值？邱永臣觉得，兄弟姐妹们对爱情的定量分析都没有搞清楚，虽然官方的说法是：『谈爱情，只应该交一个朋友』，但是如果恋爱中只有两个人，没有谈成三边形四边形多边形甚至立方体，那么世界上99%的小说和影视作品就都没有了。
有那不开眼的同学问：”邱永臣你到底是想谈两个三个还是四个五个女朋友呀？”
邱永臣心想，算了，我对艺术性的恋爱关系已经不感兴趣，自己也到了法定结婚年龄，可以光明正大地结婚了，到时候再说吧，眼前还是哄好自己女朋友要紧哪…

4.一个大男人扎到一堆不懂事的女孩子里边，唱道：
“拥抱真爱让世界更美，敞开胸怀让理想实现”，
这不是流氓是什么？其实很多歌都很流氓，但天下很多人为之倾倒。
“是不是黑夜里没来陪你相依偎 你伤悲为了谁 让我来开启你的心扉”
这不是乘着别人内心空虚勾引人吗？许多唱歌的流氓干了许多坏事，毁了别人之后一点悔改的情绪都没有。

5.大聪伟岸的身躯和傲人的胸围，掩盖不了他体内略微龌蹉的心。

6.和大陆吃饭时，谈起男女婚姻，大陆说：“很多东西并不是像电视剧里那么浪漫，人生很多事情没有想象中那么美好，相识一段时间，嬉戏打闹，然后相恋，甜甜蜜蜜一段时间，有吵有闹，历经重重挫折，接着结婚买车买房生孩子，最后生老病死”。邱永臣想：“咦，这话听起来有点耳熟？哦，原来自己也曾说过类似的话。不过，也管不了那么多了，眼前还是吃饭要紧哪。再说了，女朋友不嫁给自己的话，指不定她以后会嫁给什么挫男，还是吸烟喝酒赌博眼高手低自以为是的那种，到那种地步，自己真是心都会流血。”

7.深蓝色的天空中倒挂着一头血红的金乌。

8.小鱼用很快的语速问：“大鱼大鱼你喜欢吃什么样的小鱼？”
大鱼也快速回答道：“我喜欢吃说话快的小鱼”
于是小鱼学着树懒说话：“哦~那~我~知~道~了~”
